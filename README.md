# Iris Flower Classifier

Мой первый проект по машинному обучению.  
Модель предсказывает вид ириса по размерам чашелистика и лепестка.

## Точность
- **100%** на тестовых данных (из-за простоты задачи)

## Как запустить
1. Установи Python
2. Выполни: `python iris_ml.py`

## Используемые технологии
- Python
- scikit-learn
- Random Forest
